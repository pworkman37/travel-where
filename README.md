## User Manual
Input Constraints:
- Origin Airport Code
- Destination Airport Code
- Flight Date mm/dd/yyyy
- Passengers must be =>1
- Enter shelter city name (Any City)
- Shelter Check in Date mm/dd/yyyy
- Shelter Check out Date mm/dd/yyyy 
- Guest must be =>1
